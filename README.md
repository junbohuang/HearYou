# HearYou
Emotion Detection from Speech

# Preacquisites
- python 3.5 +
- tensorflow 1.4 +
- numpy, matplotlib, sklearn

# Data
This project uses the Interactive Emotional Dyadic Motion Capture (IEMOCAP) database.
